# Password Strength Analyzer

## Overview
This project is a cybersecurity-based Password Strength Analyzer developed using Python.

The system evaluates password strength based on:
- Password length
- Uppercase and lowercase letters
- Numbers
- Special characters
- Common password patterns

It also prevents password reuse and securely stores passwords using SHA-256 hashing.

---

## Features
- Password strength analysis
- Weak/Medium/Strong classification
- Strong password suggestions
- Password reuse detection
- SHA-256 password hashing
